# React Social (Stoped)

## Cтек:
- React
- Redux
- react-router-dom
- react-redux
- redux-thunk
##### Стили: 
- Css in Js ( Styled Components )
- SASS ( *в крайних случаях* )
##### Формы:
- Formik
- yup
##### Иконки:
- react-fontawesome
##### API:
- axios


## Установка  и запуск
Установите все зависимости

`$ npm install`

Запустите проект 

`$ npm start`
